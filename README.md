# search-subdomains
search subdomains 
este es un buscador de subdominios para ejecutarlo en la terminal pones: python pratica.py (la web que quieres analizar ) y (la ruta de tu biblioteca con subdominios)
por ejemplo: 
python pratica.py http://httpbin.org /usr/share/dnsrecon/dnsrecon/data/subdomains-top1mil-20000.txt
